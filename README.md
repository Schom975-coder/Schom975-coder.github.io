# Portfolio site

Static personal portfolio: single HTML page, shared CSS and JavaScript, and assets for images. No build step.

## Files

| Path | Purpose |
|------|---------|
| `index.html` | Page structure and copy (hero, about, skills, experience, projects, certifications, education, contact). |
| `assets/css/style.css` | Layout, theme (CSS variables at the top), and component styles. |
| `assets/js/main.js` | Navigation, scroll behavior, animations, and small UI interactions. |
| `assets/images/` | Images such as profile photo, project screenshots, optional social preview. |
| `.github/workflows/deploy.yml` | CI workflow used when hosting on GitHub Pages. |
